# ExpMan Dashboard Components

This directory contains reusable Leptos components used throughout the ExpMan dashboard.

## Key Components

- **Charts**: Line and scalar charts for metric visualization.
- **RunsTable**: Interactive data table for comparing experimental runs.
- **Artifacts**: Browsing and previewing files associated with runs.
- **Interactive**: Embedded Jupyter Notebook integration.
- **Console**: Live log streaming view.
- **Zoom**: Utilities for chart interaction.
