pub fn cmd_import(dir: PathBuf, input: PathBuf) -> Result<()>
Expand description

Import TensorBoard event logs into an expman experiment.

Reads scalar summaries from tfevents files in the given input directory (or a single event file) and creates a new expman run under dir/<input_basename>.

§Arguments

  • dir - Base experiments directory (e.g. ./experiments)
  • input - Path to a TensorBoard log directory or a single tfevents file

§Errors

Returns an error if the input path doesn’t exist, no tfevents file is found, or the event file cannot be parsed.
